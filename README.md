The Web Intelligence Layer (Dashboard)
A cloud-ready control interface built with Next.js that communicates with the robot via the rosbridge_suite (JSON API). This application serves as the command center, allowing for remote configuration, trajectory visualization, and real-time telemetry monitoring of motor health and cutting progress.

Key Tech: Next.js, WebSockets, rosbridge_suite, JSON API.
